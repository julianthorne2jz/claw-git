# claw-git

Git helper for AI agents — quick status, smart commits, branch management

## Usage

```bash
# Available commands
claw-git save
claw-git pop
claw-git list
claw-git drop
claw-git quick
claw-git q
claw-git commit
claw-git c
claw-git push
claw-git pull
```

## Options

- `--is-inside-work-tree`
- `--show-current`
- `--porcelain`
- `--short`
- `--format`
- `--quiet`
- `--count`
- `--cached`
- `--stat`
- `--force-with-lease`

## When to Use

Use this tool when you need to git helper for ai agents — quick status, smart commits, branch management.

## Notes

- Run `claw-git --help` for full usage information
